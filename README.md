# SillyTavern 快捷工具栏

*一个功能丰富的 SillyTavern 输入辅助插件，提供快捷输入、导航跳转、消息管理、工具栏定制等实用功能。*

## 功能一览

### 文本编辑

- **撤回 / 重做**：支持多步撤回和重做，最多保留 50 步历史记录
- **选中模式**：移动端文本选择辅助，点击锚定起点后再点击另一位置即可选中两点之间的文本
- **成对符号插入**：一键插入 `**`、`""`、`()`等成对符号，自动将光标定位在中间；如果已选中文本，会自动用符号包裹
- **快速换行**：自动移动到当前行末尾并插入换行符
- **插入标记**：快速插入 `{{user}}` 和 `{{char}}` 宏
- **查找替换**：在输入框或正在编辑中的消息里查找和替换文本，支持 Enter / Shift+Enter 跳转、Esc 关闭

### 导航功能

- **跳转聊天顶部 / 底部**：一键滚动到聊天最顶端或最底端
- **跳转 AI 消息顶部**：滚动到最新一条 AI 回复的顶部，方便从头阅读长回复
- **上 / 下一条 AI 消息**：在 AI 消息之间快速跳转
- **跳转指定楼层**：输入楼层号直接跳转
- **翻页模式**：开启后导航按钮变为翻页操作；移动端点击聊天区域上半部分向上翻页、下半部分向下翻页；支持音量键翻页（需安装 Key Mapper），双击音量上键跳到最新 AI 消息顶部，双击音量下键跳到聊天底部
- **自动滚动**：以设定速度自动向下滚动，适合阅读长文；用户手动滚动时暂停，2 秒后恢复

### 消息操作

- **删除最后消息**：快速删除最后一条消息（支持确认弹窗）
- **删除当前备选**：删除最后一条消息的当前备选回复
- **继续回复**：让 AI 继续生成当前回复
- **重新生成**：重新生成最后一条 AI 回复
- **QR 助手面板**：一键打开 Quick Reply 助手面板（需安装 QR 助手插件）[QR 助手](https://github.com/uhhhh15/QR.git)

### 消息隐藏管理

管理聊天消息在 AI 提示词中的可见性，隐藏的消息不会发送给 AI：

- 隐藏 / 显示指定楼层或范围
- 只保留最近 N 条消息可见
- 一键隐藏全部 / 显示全部
- 实时显示当前隐藏状态

### 自定义内容

- 添加任意自定义插入内容，点击按钮即可插入到输入框
- 支持短符号、宏标签、常用短语、整段模板文本（多行段落）
- 自定义按钮显示文字或 Font Awesome 图标（长段落建议配简短显示或图标）
- 可设置插入后的光标位置（开头、中间、结尾或自定义偏移，方便定位到需要补充的位置）

### 按钮分组

- 将多个按钮收纳进文件夹，工具栏只显示一个折叠按钮
- 点击文件夹按钮展开下拉菜单
- 支持自定义文件夹名称和图标
- 按钮可在文件夹之间自由拖拽移动
- 文件夹内按钮支持拖拽排序
- 文件夹内所有按钮被禁用或被悬浮面板占用时，文件夹按钮自动隐藏

### 工具栏定制

- 勾选控制每个按钮的显示 / 隐藏
- 拖拽调整按钮和文件夹的显示顺序
- 支持固定展开模式（无需聚焦输入框）
- **双栏模式**：工具栏分为符号输入栏和功能按钮栏，两行各自独立横向滚动，可调整上下顺序

### 悬浮面板

- **悬浮球模式**：可拖拽的悬浮球，点击展开功能面板
- **固定面板模式**：常驻显示的可拖拽功能面板
- 自定义悬浮球外观（图片 / GIF、大小、圆形 / 方形、透明背景）
- 支持展开态和收起态使用不同图片
- 面板方向可选竖向或横向
- 面板按钮支持拖拽排序和自定义大小
- **自动隐藏**：点击聊天区域显示 / 隐藏，点击其他区域自动收起
- **键盘自适应**：移动端键盘弹起时，面板自动避开酒馆顶栏并在必要时支持滚动
- 放入悬浮面板的按钮不会在主工具栏重复显示

### 辅助功能

- **非流自动跳转**：非流式模式下 AI 生成完毕自动滚动到该条消息顶部
- **续写时锁定滚动**：使用「继续回复」续写期间锁定聊天滚动位置，方便边看边续写；普通生成、重新生成、切换备选等其他场景不受影响；手动滚动可解除锁定
- **删除确认**：删除类操作可开启弹窗确认防止误操作
- **自动滚动速度**：可调节自动滚动的速度
- **美化指南**：内置提示词生成器，复制给 AI 即可生成匹配当前主题的工具栏美化 CSS
